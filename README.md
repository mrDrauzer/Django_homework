## 🛠 Запуск проекта
### 1. Клонировать репозиторий
git clone <repository_url>
cd Django_homework

text

### 2. Скопировать и настроить переменные окружения
cp .env.sample .env

Отредактируйте .env файл своими значениями (включая настройки CACHE)
text

### 3. Установить зависимости
poetry install
poetry shell

# Не забудьте установить Redis!
# pip install redis

text

### 4. Запустить Redis
redis-server

text

### 5. Применить миграции
python manage.py migrate

text

### 6. Создать суперпользователя
python manage.py createsuperuser

### 7. Запустить сервер
python manage.py runserver

text

При запуске вы увидите:
======================================================================
🚀 Запуск Django Development Server
📍 Доступные URL-адреса:

Главная страница: http://127.0.0.1:8000/

Блог: http://127.0.0.1:8000/blog/

Админ-панель: http://127.0.0.1:8000/admin/

Контакты: http://127.0.0.1:8000/contacts/

💡 Для остановки сервера нажмите CTRL+C
